# NAME
docker-volumes-create - Create a new volume

# SYNOPSIS
**docker volumes create**
[**-n**|**--name**]
[**-p**|**--path**]
[**-m**|**--mode**]

# OPTIONS
**-n**, **--name**=""
   Set volume's name.

**-p**, **--path**=""
   Set volume's path.

**-m**, **--mode**=""
   Set volume's read/write mode.
