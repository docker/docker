============================================================================
``cp`` -- Copy files/folders from the containers filesystem to the host path
============================================================================

::

   Usage: docker cp [-h] CONTAINER:RESOURCE HOSTPATH

   Copy files/folders from the RESOURCE to the HOSTPATH

    -h, --help  Display this help
