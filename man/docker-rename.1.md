# NAME
docker-rename - Rename a container

# SYNOPSIS
**docker rename**
[**-D**|**--debug**[=*false*]]
[**--help**]/
OLD_NAME NEW_NAME

# OPTIONS
**-D**, **--debug**=*true*|*false*
   Enable debug mode. Default is false.

**--help**
   Print usage statement
